# Iconocracy Corpus Explorer

Read-only public explorer for the frozen `warholana/iconocracy-corpus` dataset.

## Contract

- No writeback
- No cataloguing
- No coding workflow
- No LLM dependency

The app reads `corpus-data.json` and, when available, `release.json` from the
published Hugging Face dataset snapshot.
