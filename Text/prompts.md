# Prompts para Pesquisa Iconográfica

This section outlines various prompts designed to guide research activities within the Iconocracia project. These prompts categorize tasks related to corpus management, iconographic coding, theoretical analysis, and data synthesis.

## Categories:

- **Corpus Management:** For registering new items, managing data, and ensuring consistency.
- **Iconographic Coding:** For detailed analysis of images using frameworks like IconoCode.
- **Indicators:** For applying specific analytical metrics (e.g., the 10 indicators of purification).
- **Atlas Composition:** For building the visual Atlas by selecting and arranging images.
- **Writing Assistance:** For drafting sections of the thesis chapters.
- **Theoretical Articulation:** For developing and connecting core theoretical concepts.
- **Infrastructure:** For data synchronization tasks (e.g., GitHub and Notion).
- **Obsidian Notes:** For generating atomic Zettelkasten-style notes.
- **Bibliography:** For formatting citations according to ABNT standards and verifying references.
- **Digital Archive Search:** For querying image archives like Gallica, LoC, etc.
