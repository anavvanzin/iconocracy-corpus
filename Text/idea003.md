# idea:003 — IMES

## Description

Sintetizar atlas-argumento + campo cartográfico + LPAI-capta em um dispositivo metodológico estratificado que opera simultaneamente sobre o mesmo corpus. A nova unidade de análise é o **regime visual** — o conjunto de práticas, suportes e posições de campo que constituem uma configuração iconocrática.

## Motivation

Nenhum dos dois métodos isolados (atlas ou score) sustenta a tese. A síntese deve cancelar a premissa de unicidade metodológica, preservar o que cada método faz bem, e elevar para uma nova arquitetura onde três camadas operam simultaneamente.

## Expected outcome

Tríade operacional:
1. **Atlas-Argumento** (primária): pranchas warburguianas como demonstração historiográfica.
2. **Campo Cartográfico** (relacional): mapeamento de posições no campo jurídico-iconográfico.
3. **LPAI-capta** (instrumental): triagem e documentação de ausência.

## Risks

- **Complexidade**: três camadas = triplo trabalho. Viável no cronograma doutoral?
- **Circularidade**: as camadas se referem mutuamente sem hierarquia clara. Como navegar?
- **Banca**: metodologia triádica exige que a banca aprenda a ler algo novo.

## Mitigations

- Piloto de 10 pranchas para testar viabilidade das três camadas.
- Documentar a relação entre camadas explicitamente no capítulo metodológico.
- Preparar "guia de leitura" para banca: como ler uma prancha IMES.

## Notes

Esta é a síntese da Fase 5 da dialética atlas vs. score. Aguardando validação pelos monges (Fase 6).

Failure modes identificados na avaliação crítica:
- Confirmation bias: atlas como "camada primária" pode refletir inclinação da pesquisadora.
- HARKing visual: pranchas montadas post-hoc.
- Viabilidade da Camada 2: dados de campo podem não existir para todos os 165 registros.
- Confiabilidade inter-avaliador do LPAI-capta ainda não testada.
